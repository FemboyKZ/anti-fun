
# Anti fun plugins

Plugin breaks all func_breakable entities on command, lists details on broken entities when ran as debug.

## CS:GO

Requires SM1.11+

`sm_nofun` / `sm_breakall`

`sm_nofun_debug` - `sm_breakall_debug`

## CS2

Requires CS#

`css_nofun` / `css_breakall`

`css_nofun_debug` - `css_breakall_debug`
